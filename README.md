# Navi_Bus
## RV University, Semester 2 Final project
## Bus Tracking System.

## Component requirements
- LoRa Ra-02        --> 2 pieces
- Neo gps module    --> 1 piece
- Arduino uno       --> 2 pieces
- Jumper Wires      --> 1 pack
- Bread Board       --> 2 pieces
- Node MCU          --> 1 piece


## Software Requirements
- Arduino IDE (on at least 2 systems)
- TinyGPS+ module library
- SPI module library 
- LoRa module library
